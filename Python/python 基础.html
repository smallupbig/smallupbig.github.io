# python 基础

# 数据类型和变量

  1. 整数

    可以处理任意大小的整数，包括负整数

  2. 浮点数

  3. 字符串

    字符串是以单引号或双引号括起来的任意文本

    • 转义字符：

      • \n:换行
      • \t:制表符
      • \:\

    • 允许使用 r'' 表示内部的字符默认不转义

    • 如果字符串内部有很多换行，用 \n 写在一行不好阅读，可以使用 '''内容''' 的格式，相当于 C# 中的 @ 符号

  4. 布尔值

    一个布尔值只有 True 和 False 两种值，注意大小写 布尔值可以用 not and or 计算

  5. 空值

    None

  6. 变量

    变量名必须是大小写英文、数字和 _ 的组合，且不能用字母开头

# 运算符

  1. / 整除：浮点数
  2. // 整除：永远为整数
  3. % 取余

# 字符串

字符串以 Unicode 编码，一个字符对应若干个字节 ord():获取字符的整数表示 chr():把编码转换成对应的字符

str 转 bytes:

  • bytes 类型的数据用带 b 前缀的单引号或双引号表示： x = b'ABC'
  • bytes 的每个字符都只占用一个字节
  • 'ABC'.encode('ascii') //unicode 转 ascii
  • '中文'.encode('utf-8') //unicode 转 utf-8
  • 含有中文的 str 无法用 ascii 编码，因为中文编码的范围超过了 ascii 编码的范围
  • 如果 bytes 中只有一小部分无效的字节，可以传入 decode('utf-8', errors='ignore') 忽略错误的字节
  • 长度:len('ABC')

格式化： 与 c 相似: 'hello, %s' % 'world'

  • %d 整数
  • %f 浮点数
  • %s 字符串
  • %x 十六进制整数
  • format(): 'hello, {0}'.format('world')

# list

python 内置的一种数据类型: a = ['a', 'b']

  • append('c'):追加元素
  • insert(1, 'c'):插入元素
  • pop():删除 list 末尾的元素
  • pop(i):删除指定位置的元素

# 循环

for 循环：


a = ['1', '2']

for x in a:
    print(x)

while 循环：


sum = 0;
n = 10;
while n > 0: 
    sum = sum + n
    n = n - 1

print(sum)

# dict

键值对


d = {'a' : 90, 'b' : 80}

print(d['a'])
  • 'a' in d :判断 key 是否存在
  • d.get('a') :如果 key 不存在，返回 None，或者自己指定返回的值
  • pop('a') :删除一个 key

# set

和 dict 类似，但不存储 value


s = set(['a', 'b'])
s.add('c')
s.remove('c')
print(s)

# 函数

自定义函数：


def my_funcation(x):

    if not isinstance(x, (int, float)):
        raise TypeError('bad type')
    if x > 0:
        return 5
    else:
        return x

print(my_funcation('2'))

空函数：


def nop():
    pass

返回多个值： 其实只是返回一个 tuple， 接收的时候只是按位置赋予对应的值


def move(x, y, num):
    nx = x + num
    ny = y + num
    return nx, ny


print(move(1, 2, 3))

可变参数：


def calc(*numbers):
    sum = 0
    for n in numbers:
        sum = sum + n * n
    return sum

print(calc(1, 2, 3))

关键字参数：


def preson(name, age, **k):
    print(k['city'])
    print('name:', name, 'age:', age, 'other:', k)


# preson(1, 2, city='BeiJing')

extra = {'city': 'BeiJing'}
preson(1, 2, city=extra['city'], node=extra)

**k 表示把 extra 这个 dict 的所有 key-value 关键字都传入了 **k 这个参数之中，k 只是 extra 的一份拷贝，对 k 的改动不会影响 extra

命名关键字参数：


def preson(name, age, *, city, k):
    print('name:', name, 'age:', age, 'other:', k)

extra = {'city': 'BeiJing'}
def test(name, *k, city):
    print('name:', name, 'age:', k, 'city:', city)

test('test', city=extra['city'])

参数组合： 顺序：必选参数、默认参数、可变参数、命名关键字参数和关键字参数

对于任意函数，都可以通过类似 func(*args, **k) 的形式去调用它，无论它的参数是如何定义的
