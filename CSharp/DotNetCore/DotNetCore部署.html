# 守护进程 Supervisor

当我们关闭远程连接时，控制台窗口也随之关闭，为了让我们发布的项目能一直运行，安装 Supervisor 来守护我们的进程

# 安装 Supervisor

yum install python-setuptools
easy_install supervisor

# 配置

  • 创建配置文件
mkdir /etc/supervisor

echo_supervisord_conf > /etc/supervisor/supervisord.conf
  • 修改 supervisord.conf 文件，在文件最后面修改为如下
[include]
files = /etc/supervisor/conf.d/*.conf
  • 创建对项目的守护

在 /etc/supervisor/conf.d 下创建文件 MyPlat.conf

[program:MyPlat]
command=dotnet MyPlat.dll
directory=/root/web/MyPlat/
autorestart=true ;是否自动重启
stderr_logfile=/var/log/supervisord/web/MyPlat.err.log ;文件目录记得创建，不然会报错
stdout_logfile=/var/log/supervisord/web/MyPlat.out.log
environment=ASPNETCORE_ENVIRONMENT=Production ;进程环境变量
user=root
stopsignal=INT
  • 启动 supervisor 配置文件
supervisord -c /etc/supervisor/supervisord.conf
  • 查看进程

如果存在 donnet MyPlat.dll,则表示启动成功

ps -ef | grep MyPlat
  • 配置 supervisor 开机启动

创建文件 supervisord.service，

拷贝到 /usr/lib/systemd/system/supervisord.service，

执行命令 systemctl enable supervisord

执行命令：systemctl is-enabled supervisord #来验证是否为开机启动

# dservice for systemd (CentOS 7.0+)
# by ET-CS (https://github.com/ET-CS)
[Unit]
Description=Supervisor daemon

[Service]
Type=forking
ExecStart=/usr/bin/supervisord -c /etc/supervisor/supervisord.conf
ExecStop=/usr/bin/supervisorctl shutdown
ExecReload=/usr/bin/supervisorctl reload
KillMode=process
Restart=on-failure
RestartSec=42s

[Install]
WantedBy=multi-user.target

# 常用命令

  • 重启配置文件
supervisorctl reload

# Nginx

当我们部署多个项目的时候，不可能部署一个项目，服务器便要开启一个端口，这时候安装 Nginx 来负责 80 端口的监控，然后负责对多个项目的分发。

# 安装 Nginx

curl -o  nginx.rpm http://nginx.org/packages/centos/7/noarch/RPMS/nginx-release-centos-7-0.el7.ngx.noarch.rpm
rpm -ivh nginx.rpm
yum install nginx

# 配置对项目的转发

将 /etc/nginx/conf.d/default.conf 的内容替换如下，然后重启 Nginx

server {
    listen       80;
    server_name  www.baidu.com ;你的网站地址

    location / {
        proxy_pass http://localhost:8080/ ;本地访问地址
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection keep-alive;
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
    }
}

# Nginx 常用命令

  • 启动
systemctl start nginx
  • 开机启动
systemctl enable nginx
  • 重启配置
nginx -s reload
