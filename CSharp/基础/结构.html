# 结构

# 定义

  1. 与类相似，也有数据成员和函数成员
  2. 区别
    • 类是引用类型,结构是值类型
    • 结构隐式密封的，不能被派生
    • 结构可以有实例构造函数和静态构造函数，但是没有析构函数

# 值类型

  • 变量不能为 null
public struct Student{
        public string name;
        public int age;

        public void SetName(string name){
            this.name = name;
        }
    }

    static void Main(string[] args) {
        Student student = null; //出错
    }
}

# 实例构造函数

  1. 语言隐式地为了每个结构提供无参数的构造函数，值成员为默认值。
  2. 预定义的无参构造函数对每个结构都存在，且不能删除或重定义
  3. 也可以不用 new 运算符创建结构的实例
    • 在显式设置数据成员之后，才能使用它们的值
    • 在对所有数据成员赋值之后，才能调用任何函数成员

static void Main(string[] args) {
    SStudent student;
    student.age = 1;
    //显式设置数据成员之后，才可以使用它们的值
    Console.WriteLine(student.age);//可以使用
    Console.WriteLine(student.name);//出错
    //在对所有数据成员赋值之后，才能调用任何函数成员
    student.SetName("name");//出错
    student.name = "gu";
    student.SetName("name");//可以使用
}

# 静态构造函数

  1. 与类相似，结构的静态构造函数创建并初始化静态数据成员，而且不能引用实例成员。
  2. 调用静态构造函数（满足其一即可）：
    • 调用显式声明的构造函数
    • 引用结构的静态成员

# 字段不可以初始化语句

public struct Student{
    public string name = "gu";//出错
    public int age;

    public void SetName(string name){
        this.name = name;
    }
}

# 作为参数和返回值

  • 返回值：将创建它的副本并从函数成员返回
  • 值参数：将创建实参结构的副本。该副本用于方法的执行中
  • ref 和 out 参数：传入的是该结构的引用，这样可以修改其数据成员
