# 枚举

# 定义

  1. 值类型：直接存储数据
  2. 只有一种类型的成员：命名的整数型变量

# 特点

  1. 每个枚举成员都被赋予一个底层类型的常量值
  2. 默认情况下，第一个成员被赋值为0，后续成员的值比以前的值多1
  3. 枚举名 : [类型名]，就可以使用int以外的整数类型
  4. 成员常量的值可以是底层类型的任何值，不能有重复的名字，但是可以有重复的值。

# 赋值规则

  1. 未被显式赋值的成员变量会比前一个成员多1

# 示例

  • 声明一个枚举类
enum Hero {
    Pig,    //默认为0
    Dog,    //1
    Egg     //2
}
  • 显式声明一个枚举类

enum Hero {
    Pig = 10,
    Dog = 15,
    Egg = 15 //可以有重复的值
}
