# MySql 安裝

# 环境

系统： Windows

# 下载

  1. 下载解压包
  2. 解压到 C 盘

# 配置环境

  1. 添加系统变量
  2. 在 Path 中添加 %Mysql_Home%

# 生成 data 文件

  • 执行命令
mysqld --initialize-insecure --user=mysql
  • 初始化 mysql
mysqld -install
  • 启动服务
net start mysql
